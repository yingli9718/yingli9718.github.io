Capstone Project

 

The aim of this project is to analyze the publicly available data from Airbnb to develop two solutions/tools:
1) Booking Optimizer model to predict booking for a listing, which is to provide insights and recommendations for the listing host to take actions to optimize his books and thus revenue.
2) Listing Price Optimizer model to predict the “right price” for each of the listing, which is to help the listing hosts to maximize the revenues.
[GitHub repo] [Report]
